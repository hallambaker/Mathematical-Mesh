

# calendar

~~~~
<div="helptext">
<over>
calendar    Manage calendar catalogs connected to an account
    add   Add calendar entry
    delete   Delete calendar entry
    dump   List calendar entries
    get   Lookup calendar entry
    import   Add calendar entry from file
<over>
</div>
~~~~
