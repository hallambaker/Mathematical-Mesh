Goedel.ASN Namespace

Support library for ASN.1 encoding and decoding using the ASN tool.
Classes
  ClassDescription
ASN
Utility class containing static methods.
ASNDecodingException
An error occurred in the decoding of presumed ASN.1 binary data.
Buffer
Buffer class for assembling ASN.1 output data in DER encoding.
Constants
Constants used for encoding and decoding ASN.1 data
DecodeBuffer
Buffer used to decode ASN1 data
ExpectedInteger
An integer was expected.
ExpectedSequence
A sequence of items was expected.
Implementation
The data could not be decoded due to an implementation restriction in the decoder. This should not happen when attempting to decode legitimate inputs for the intended field of use.
IndefiniteLengthInvalid
An indefinite length was specified in a context where it is not permitted. For example, a DER encoded item.
IntegerOverflow
The size of an integer exceeds the implementation limit. This is unlikely to occur in normal use.
InvalidLength
A length specification was invalid.
LengthExceedsInput
The declared length of an item exceeds the available data. This typically happens if the data item has been truncated or a malicious payload is attempting a buffer overflow attack.
LengthExceedsStructure
The length of an inner length encoded item exceeds that of the enclosing item. This means that either the data is not ASN.1, the data has been corrupted or is a malicious payload intended to perform a buffer overflow attack.
Root
Root class for all ASN.1 backing structures
UnExpectedData
Data was encountered in an unexpected location.
