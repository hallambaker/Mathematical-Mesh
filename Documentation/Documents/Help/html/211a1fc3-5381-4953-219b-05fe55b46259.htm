Classes

DebugLocalSession
Direct connection between client and service host with messages encoded and decoded from JSON. For use in debugging issues that might be the result of JSON encoding issues and to collect samples for documentation.
TraceDictionary
The trace dictionary class is used for capturing a set of protocol messages for use in documentation. The functions supported include capture of the message objects and presenting them in full or with various levels of redaction.
TraceMessage
Represents a single trace message
TracePoint
